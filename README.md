1) Практическое задание по теме «Android Studio и Git»
Цель задания
Научиться создавать проект в Android Studio, запускать приложения на эмуляторе, а также научиться создавать свой Git репозиторий и пушить в него изменения.

Что нужно сделать
1. Создать проект по шаблону в Android Studio.
2. Создать эмулятор и запустить приложение на нём
3. Проинициализировать свой git repository
3.1. Зарегистрироваться на  github.com
3.2. Создать (или добавить) новый SSH ключ
3.3 Создать репозиторий
4. Cделать первый коммит в GitHub




2) Практическое задание по теме «Создание UI в Android»
   Цель задания
   Сверстать экран со списком приложений и настроить переход в карточку приложения.

Что нужно сделать
1. Сверстать экран по макету (список приложений)
   — Верстка в Jetpack Compose.
   — Отступы — «на глаз» (ориентируемся на визуальное соответствие макету, без пиксель-перфекта).
   — Данные захардкодить (пример структуры данных и практической части — в репозитории).
2. Добавить навигацию по клику
   — По клику на элемент списка открывать экран карточки приложения из практической части.
   — Карточку приложения взять из практического репозитория (используем готовый экран/модуль из практики, не придумываем заново).
3. Навигация
   — Для навигации использовать Jetpack Compose Navigation (NavHost / composable / navController).
